Este projeto consiste na criação de um formulário HTML com diferentes tipos de campos, como texto, botões de opção (radio) e listas suspensas (select).

O objetivo principal é praticar:
- Estruturação de formulários em HTML
- Organização de código
- Uso de Git e GitHub
- Fluxo de desenvolvimento com branches e Pull Requests

Cada funcionalidade do formulário é desenvolvida em uma branch separada, seguindo boas práticas de versionamento utilizadas no mercado.
